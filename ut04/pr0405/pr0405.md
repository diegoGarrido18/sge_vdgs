## Ejercicio 1 - Filtrado de una lista de números

## Ejercicio 2 - Mapeo de temperaturas

```python
celsius = [0, 20, 37, 100]
fahrenheit = list(map(lambda c: c * 9/5 + 32, celsius))

print(fahrenheit)
```

## Ejercicio 4 - Palabras con cierta longitud
